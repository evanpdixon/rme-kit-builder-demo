# RME Kit Builder Demo

Static HTML demo/reference pages for the kit builder wizard. Pre-built HTML snapshots showing the UI at different states for testing and documentation.

## Tech Stack
- HTML, JavaScript (kit-builder.js embedded)

## Structure
- `index.html` — Main demo with full wizard
- `backlog.html` — To-do list view
- `branded.html` — Branded variant
- `daily.html` — Daily use view
- `kit-builder-admin.html` — Admin configuration UI

## Notes
- Standalone HTML demos, no server required
- Config stored in localStorage (fallback mode)
- Used for snapshot-based documentation and offline testing
